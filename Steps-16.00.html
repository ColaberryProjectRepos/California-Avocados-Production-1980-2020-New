Creating textbox for the KPI Card

 1.    In the report view, Select Insert ribbon>Text box.
        1a. In the Formatting Tab-->General--> Properties; Height = 132, Width = 200,                       Position: Horizontal = 1, Vertical = 189, Background: On, Color = White, 10%                    Darker.
        1b. Please copy (ctrl+c) and paste (ctrl+v) text box three times before you                      move on to house KPI cards or insert text from the ribbon and follow the                measurements below.
       1c. In the Formatting Tab-->General--> Properties; Height = 132, Width = 200,                    Position: Horizontal = 1, Vertical = 321, Background: On, Color = White, 10%                    Darker.
       1d. In the Formatting Tab-->General--> Properties; Height = 135, Width = 200,                       Position: Horizontal = 1, Vertical = 453, Background: On, Color = White, 10%                   Darker.
       1e. In the Formatting Tab-->General--> Properties; Height = 132, Width = 200,                     Position: Horizontal = 1, Vertical = 588, Background: On, Color = White, 10%                   Darker.
