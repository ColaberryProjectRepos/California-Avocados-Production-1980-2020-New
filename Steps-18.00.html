Adding Sub KPIs. (Sales YoY%, Production YoY%, Harvest YoY%, Price YoY%) 

1.   Click on an empty space on the canvas, and from the visualizations pane select Visual Card.
1a.   Drag the Sales YoY % Sign measure to the field value box.
1b. In the Formatting Tab-->Visual-->Callout value: Font = Din, size = 16, Color = E66C37, Display Unit: Auto, Value decimal Places = Auto, Category Label = off. 
1c. In the Formatting Tab-->General-->Properties-->Size: Height = 43, Width = 86, Position: Horizontal = 75, Vertical = 271, Title: Off, Effect: Background: On, Color = White, 10% Darker.
