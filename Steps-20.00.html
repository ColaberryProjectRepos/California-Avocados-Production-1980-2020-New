Adding Sales by Year Chart



1.    Click on an empty space on the canvas and from the visualizations pane select Line and Clustered Column Chart.

2.    Drag the Year to the X-axis and the Total value on the Column y-axis. Drage all the measures on Line Y-Axis (Harvest YoY%, Sales YoY%, Production YoY%, Price YoY%)

3. Drag all the above measures into the Tooltips field as well. (Harvest YoY%, Sales YoY%, Production YoY%, Price YoY%)

4. In the Formatting Tab-->x-axis: On, Font = Segoe UI Bold, size = 9, Color = Black, 40% Lighter. Title: off  

y-axis: On, Font = Segoe UI Bold, size = 9, Color = Black, 40% Lighter. Title: Off. Secondary y-axis: Off.

Legend: On, Position = Top left, Text: Font = Segoe UI Bold, size = 10, Color = Black, 40% Lighter. Title: On.

Gridlines: Horizontal: On style = Dotted, Color = White, 20% darker, width = 1, Vertical: On style = dotted, Color = White, 20% darker, width = 1

Line Colors:  Price YoY% = #E044a7, Production YoY% = #D9B300, Sales YoY% = #E66c37, Harvest YoY% = #12239E, Shaded area: Off. 

Column: Color = #257f54. 

Data labels: On. Data labels for all lines turned off except Price YoY%

5. In the Formatting Tab-->General-->Properties-->Size: Height = 343, Width = 604, Position: Horizontal = 201, Vertical = 57, Title: Sales by Year, Font = Din, Size = 14, Color = #257f54, Background: #D9B300 , Center, Effect: Background: Off. 

Tooltips: On, Font = Segoe UI, Size = 9, Label Color = c8c6c4, Value Color = White, 10% darker. and Background: Black.
