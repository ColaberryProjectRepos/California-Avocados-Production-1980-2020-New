Adding Year Slicer



1.   In the Report View, open the visualization pane, then Select Slicer from the                      visualization
2.   In the fields, open Calendar Table, and drag the Year.
       2a. In the Formatting Tab-->Visual-->Slicer settings = In Between, Slicer Header =                Off, Values: Font = Segoe UI Bold, Font = 9, Font Color = Black, Background =                  off, Slider: On, Color = Black
         2b. In the Formatting Tab-->General-->Properties-->Size: Height = 132, Width =                   200, Position: Horizontal = 1, Vertical = 57, Title: Year, Font = Din, Size = 14,                      Horizontal Alignment = Center, Text Color = #257F54, Background color:                          #e1c233, Effect: Background: On, Color = White, 10% Darker.
