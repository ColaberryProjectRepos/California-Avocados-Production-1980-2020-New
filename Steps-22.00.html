Creating Production by County Chart



1. Create a County state column by right-clicking on the Cali_Avocados table. Paste the below formula. 

County State = (cali_avocados[ County]) & "-" & CONVERT("California",STRING)
 
Please see the below gifs for guidance.


Click on an empty space on the canvas, and from the visualizations pane select Scattered Chart.

2.   Drag County State into Values.  Drag Production to the X-axis and Sales YoY% on Column y-axis. Drag total Production into the size field. 

3. Drage measures on Tooltips.  (Harvest YoY%, Price YoY%)

4. In the Formatting Tab-->x-axis: On, Font = Segoe UI Bold, size = 9, Color = Black, 40% Lighter. Title: on, Font = Din, 12. Color = #D9B300

y-axis: On, Font = Segoe UI Bold, size = 9, Color = Black, 40% Lighter. Title: Off. Secondary y-axis: Off, Title: on, Font = Din, 12. Color = #e66c37

Legend: On, Position = Top left, Text: Font = Segoe UI Bold, size = 10, Color = Black, 40% Lighter. Title: On, Title Text: Harvested Acres 

Gridlines: Horizontal: On style = Dotted, Color = White, 20% darker, width = 1, Vertical: On style = dotted, Color = White, 20% darker, width = 1

Markers: Color: Click on Fx Button and it will take you to a new screen. Format Style: Gradient, What field should we base on: Harvested Acres (Cali_Avocados table),  click on add a middle color 

Lowest Value Color = #a0daff, Theme color 1, 60% lighter

Middle-Value Color = #41a4ff, Theme Color 1, 20% lighter

Highest Value Color = #094780, Theme Color 1, 50% Darker and click OK. 

Category Labels = On, Font: = Segoe UI Bold, size = 9, Color = Black, 40% Lighter

5. In the Formatting Tab-->General-->Properties-->Size: Height = 322, Width = 606, Position: Horizontal = 200, Vertical = 398, Title: Production by County, Font = Din, Size = 14, Color = #257f54, Background: #D9B300 , Center, Effect: Background: Off. 

Tooltips: On, Font = Segoe UI, Size = 9, Label Color = c8c6c4, Value Color = White, 10% darker, and Background: Black.
