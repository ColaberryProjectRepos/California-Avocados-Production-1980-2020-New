Creating DAX Measures
 
1. Left-click on Measure_Table and select new measure.
2.    Type or paste your measure and click on  sign and save it. 
 Please see the following gif for guidance.


1-  For Price
 
 .avg.Price current year = CALCULATE (AVERAGEX(cali_avocados, cali_avocados[ Price P/U]))
 
. avg.price last year = CALCULATE(AVERAGEX(cali_avocados, cali_avocados[ Price P/U]),DATEADD('Calendar Table'[Date],-1,YEAR))
 
Price YoY% = Var cy = [.avg.Price current year]
var ly = [.avg.price last year]
Return 
DIVIDE((cy-ly), ly )
--Format: %, Decimal Place: 2
 
Price Sign = SWITCH(TRUE(),([Price YoY%])>0,UNICHAR(9650), ([Price YoY%])<0,UNICHAR(9660)," ")
--Format: Text

Price Color = IF([Price YoY%]>0,"#1AAB40","#D6272C")
--Format: Text
 
2-  For Sales
 
Total Sales Current Year = SUM(cali_avocados[TotalValue])
 
.Sales Last Year = CALCULATE(SUM(cali_avocados[TotalValue]),DATEADD('Calandar Table'[Date],-1,YEAR))
 
Sales YoY% = DIVIDE(([Total Sales Current Year]-[.Sales Last Year]),[.Sales Last Year])
--Format: %, Decimal Place: 2

Sales YOY Color = IF([Sales YOY]>0,"#1AAB40","#D6272C")
--Format: Text
 
Sales YOY Sign = SWITCH(TRUE(),([Sales YOY])>0,UNICHAR(9650), ([Sales YOY])<0,UNICHAR(9660)," ")
--Format: Text
 
3-  For Production 
 
Total production = SUM(cali_avocados[ Production])
 
.YOY avg.Production current year = CALCULATE(AVERAGEX(cali_avocados, cali_avocados[ Production]))
 
.YOY avg.production last year = CALCULATE(AVERAGEX(cali_avocados, cali_avocados[ Production]),DATEADD('Calandar Table'[Date],-1,YEAR))
 
Production YoY% = 
Var cy = [.YOY avg.Production current year]
var ly = [.YOY avg.production last year]
Return 
DIVIDE((cy-ly), ly )
--Format: %, Decimal Place: 2
 
production YOY Color = IF([Production YoY%]>0,"#1AAB40","#D6272C")
--Format: Text
 
Production YoY Sign = SWITCH(TRUE(),([Production YoY%]*100)>0,UNICHAR(9650), ([Production YoY%]*100)<0,UNICHAR(9660)," ")
--Format: Text
 
4-  For Harvest
 
.YOY avg.Harvest current year = CALCULATE(AVERAGEX(cali_avocados, cali_avocados[ Harvested Acres]))

--Format: Text
