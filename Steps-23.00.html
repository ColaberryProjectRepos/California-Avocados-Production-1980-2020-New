 Click on an empty space on the canvas, and from the visualizations pane select Scattered Chart.

2.   Drag County State into the Location field and Drag Total Production to the Bubble size field. 

3. Drage measures on Tooltips.  (Harvest YoY%, Price YoY%, Sales YoY%, Production YoY%)

4. Map settings: Style = Road, Controls: Auto Zoom = On

5. Bubbles: Colors = Click on Fx Button and it will take you to a new screen. Format Style: Gradient, What field should we base on: County (Cali_Avocados table),  click on add a middle color 

Lowest Value Color = #f0e199

Middle Value Color = #e1c233

Highest Value Color = #a38600 and click OK. 

Category Labels = Off

6. In the Formatting Tab-->General-->Properties-->Size: Height = 322, Width = 474, Position: Horizontal = 805, Vertical = 398, Title: Price by County, Font = Din, Size = 14, Color = #257f54, Background: #D9B300 , Center, Effect: Background: Off. 

7. Insert an image from the ribbon.  In the Formatting Tab-->General-->Properties-->Size: Height = 25, Width = 153, Position: Horizontal = 817, Vertical = 609.

8. Insert the text box from the ribbon. In the Formatting Tab-->General-->Properties-->Size: Height = 27, Width = 56, Position: Horizontal = 826, Vertical = 638. Write Low in a text box. Font: Segoe UI, size = 8, Color = Black, Effects: Background Color = #f0e199

9. Insert another text box from the ribbon. In the Formatting Tab-->General-->Properties-->Size: Height = 27, Width = 52, Position: Horizontal = 917, Vertical = 638. Write High in a text box. Font: Segoe UI, size = 8, Color = Black,     Effects: Background Color = # a38600
