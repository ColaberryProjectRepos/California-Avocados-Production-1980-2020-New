Adding Indicators

1.   Click on an empty space on the canvas, and from the visualizations pane select Visual Card.
1a.   Drag the YoY% to the field value box from the measure table.
1b. In the Formatting Tab-->Visual-->Callout value: Font = Din, size = 20, Color = Click on fx, then Select Format style = Field value, what field should we base this on = YoY% Color, Click Ok.
1c. In the Formatting Tab-->General-->Properties-->Size: Height = 35, Width = 41, Position: Horizontal = 157, Vertical = 274, Title: Off, Effect: Background: On, Color = White,10% Darker, tooltips = On
 
2.   Click on an empty space on the canvas, and from the visualizations pane select Visual Card.
2a.   Drag the Production YoY Sign to the field value box from the measure table.
2b. In the Formatting Tab-->Visual-->Callout value: Font = Din, size = 20, Color = Click on fx, then Select Format style = Field value, what field should we base this on = Production YoY Color, Click Ok.
2c. In the Formatting Tab-->General-->Properties-->Size: Height = 35, Width = 41, Position: Horizontal = 157, Vertical = 411, Title: Off, Effect: Background: On, Color = White,10% Darker, tooltips = On
 
3.   Click on an empty space on the canvas, and from the visualizations, pane select Visual Card.
3a.   Drag the Harvest YoY Sign to the field value box from the measure table.
3b. In the Formatting Tab-->Visual-->Callout value: Font = Din, size = 20, Color = Click on fx, then Select Format style = Field value, what field should we base this on = Harvest YoY Color, Click Ok.
3c. In the Formatting Tab-->General-->Properties-->Size: Height = 35, Width = 41, Position: Horizontal = 157, Vertical = 538, Title: Off, Effect: Background: On, Color = White,10% Darker, tooltips = On
 
4.   Click on an empty space on the canvas, and from the visualizations pane select Visual Card.
4a.   Drag the Price Sign to the field value box from the measure table.
4b. In the Formatting Tab-->Visual-->Callout value: Font = Din, size = 20, Color = Click on fx, then Select Format style = Field value, what field should we base this on = Price Color, Click Ok.
4c. In the Formatting Tab-->General-->Properties-->Size: Height = 35, Width = 41, Position: Horizontal = 157, Vertical = 673, Title: Off, Effect: Background: On, Color = White,10% Darker, tooltips = On
